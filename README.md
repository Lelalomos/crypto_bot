# PERPOSE
#### เพื่อพัฒนาตัว bot เพื่อใช้เรียนรู้การนำเอา algorithm มาใช้ในการเทรดค่าต่างๆมากขึ้น การพัฒนาอย่างนี้สามารถนำไปต่อยอดใช้กับ หุ้น forex หรือจะเป็นการคำนวณการซื้อขายกองทุนก็สามารถทำได้
